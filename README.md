## Run Locally

Clone the project

```bash
  git clone https://github.com/ahmdchr/ISS_Al_Mokhtar.git
```

Go to the project directory

```bash
  cd ISS_Al_Mokhtar
```

Install dependencies

```bash
  pip install pygame
```

Start the server

```bash
  py ./code/main.py
```
